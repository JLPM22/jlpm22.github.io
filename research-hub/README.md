## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/`: Next.js App Router pages and API routes.
- `components/`: Reusable UI components (Sidebar, Particles, etc).
- `data/`: Local JSON/YAML data files (mock database).
- `lib/`: Utility functions.
